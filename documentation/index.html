Files structure

Item come with 3 version (plus 2 effect show) with alot of custom setting, choose your version to start by folder name below

  • static-without-parallax - image background header without parallax effect
  • static-parallax - image background header with parallax effect
  • slideshow-fade - slideshow image background header with fadein fadeout effect
  • slideshow-kenburns - slideshow image background header with kenburns effect
  • video - video background header

you can switch to any version at anytime by change the js/variable.js file

inside each version folder have the folder below

  • css - style file
  • fonts - icon font
  • img - template image
  • js - javascript file
  • php - a working contact form and subscribe form
  • index.html - main html file

Header Logo and Footer Logo

logo path at

img/header-logo.png
img/footer-logo.png

replace it to change

to style it, open css/style.css find

header-logo

.header-logo {
  position: absolute;
  height: 50px; /* logo image height */
  top: -110px; /* -(logo image height + 60px) */
  left: 0;
  right: 0;
  margin: 0 auto;
  max-width: 100%;
  z-index: 1000;
  display: inline-block;
}

height = image height

top = -(logo image height + 60px)

for example,     -(50px image height + 60px ) = -110px

 

footer logo

.footer-logo {
  height: 50px; /* logo image height */
  max-width: 100%;
  display: inline-block;
  margin-bottom: 60px;
}

change the height to logo height

Subscribe php and mailchimp



	Subscribe Mailchimp version



	item default use a php subscribe form, you can switch to mailchimp form by edit js/variable.js



	find the code below


var mailChimpVersion = false; // if false, php/sunscribe.php will do the subscribe job
var mailChimpUrl = "YOUR_MAILCHIMP_POST_URL";



	as you see, change false to true will enable mailchimp form



	1. first you need a mailchimp account, api, and list id, for more info please view http://mailchimp.com/



	2. and then find your mailchimp form id, login mailchimp, select your list, click Signforms > Embedded forms > find Copy/paste onto your site



	now search


<form action="



	you will see something like that


<form action="xxx.xxx.list-manage.com/subscribe/post?u=5afsdhfuhdsiufdba6f8802&id=4djhfdsh9"



	that link is your mailchimp post url



	open js/variable.js search


var mailChimpUrl = "YOUR_MAILCHIMP_POST_URL";



	replace YOUR_MAILCHIMP_POST_URL with your post url
